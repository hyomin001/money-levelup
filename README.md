# 💡 머니레벨업 (Money Level-Up)

> **최신 업데이트 — 🔐 4자리 PIN 로그인 보호 추가**
> - 기존에는 "이름+출생연도"만 일치하면 누구든 그 기록을 이어볼 수 있어서, 동명이인+동일 출생연도인
>   경우 서로의 가계부·자산 데이터가 섞일 수 있었습니다. 이제 회원가입 시 4자리 PIN을 함께 설정하고,
>   다음 방문부터는 이름+출생연도+PIN이 모두 일치해야 기존 기록을 이어볼 수 있습니다.
> - 이 기능 이전에 만들어진 기록은 PIN이 없는 상태이므로, 다음 로그인 때 입력한 PIN이 그대로 새 PIN으로
>   설정됩니다 (기존 사용자가 갑자기 잠기지 않도록 하는 안전한 마이그레이션).

> **이전 업데이트 — 📖 이용 가이드 / 첫 방문 튜토리얼 신설**
> - **첫 방문 시 자동 튜토리얼**: 회원가입 직후, 앱의 핵심 개념(실제 자금 vs 모의투자)부터 대시보드·
>   리스크 체험관·노후 준비·AI 코치까지 10개 섹션을 아코디언 형태로 훑어보는 안내 화면이 한 번 뜹니다.
>   "이제 시작할게요" 또는 "나중에 볼게요"로 넘어갈 수 있고, 한 번 보면 다시 자동으로 뜨지 않아요.
> - **📖 가이드 탭 (상시 열람)**: 첫 번째 탭에 항상 이용 가이드가 있어서, 나중에 헷갈리는 기능이 생기면
>   언제든 펼쳐볼 수 있어요.
> - **사이드바 "이용 가이드 다시 보기" 버튼**: 튜토리얼 화면 전체를 다시 보고 싶을 때 언제든 클릭 한 번으로
>   불러올 수 있어요.
>
> **이전 업데이트 — 🧓 노후 준비 설계 · 🎭 소비 페르소나 · 📇 리포트 카드**
> - **🧓 노후 준비 설계 (신규 탭)**: 현재 나이·자산·매월 저축액·예상 수익률/물가상승률을 입력하면
>   은퇴 시점까지의 자산 성장 곡선과 "연간 생활비의 25배(4% 인출룰 근사)" 기준 목표 노후자금을
>   나란히 비교합니다. 목표에 부족하면 필요한 월 저축액을 역산해서 알려줍니다. MZ 세대의 투자 붐뿐
>   아니라, 성인 금융교육의 핵심 축인 "노후 준비"까지 다뤄 대상 폭을 넓혔습니다.
> - **🎭 소비 페르소나**: 최근 3개월 가계부 카테고리 비중을 규칙 기반으로 분석해 "구독 중독형",
>   "카페인 러버형", "플렉스형", "안정 지향형" 등으로 태그를 붙여줍니다. 뱃지 탭에서 확인할 수 있습니다.
> - **📇 금융 리포트 카드**: 레벨·뱃지·재무 건강 점수·소비 페르소나·리스크 체험관 성적을 한 장으로
>   모은 요약 카드를 앱 안에서 미리보고, HTML 파일로 다운로드해 공모전 제출/발표 자료로 바로
>   활용할 수 있습니다.
>
> **이전 업데이트 — 🚨 리스크 체험관 신설 (실제 투자를 시작하기 전에 "위험"부터 배우는 공간)**
> - **🎣 금융사기 대응 훈련**: 리딩방 원금보장, 유사수신 P2P, 가족 사칭 메신저피싱, 코인 에어드랍 피싱,
>   대환대출 사기, 재택부업 선입금, 딥페이크 투자 광고, 지인의 몰빵 권유까지 — 실제 신고가 많은 8가지
>   유형을 가상 메시지로 재구성해, 선택에 따른 결과와 판단 기준을 즉시 피드백으로 보여줍니다. 80점
>   만점으로 채점해 "사기 방어 마스터" 등급까지 도전할 수 있습니다.
> - **🎢 레버리지 위험 시뮬레이터**: 실제/모의 자금과 완전히 분리된 가상 시드머니로 1~10배 레버리지를
>   직접 체험합니다. 같은 시세 흐름을 레버리지 유무로 나란히 비교해, 변동성이 어떻게 증폭되고 왜
>   강제 청산(로스컷)이 발생하는지 몸으로 이해하게 합니다.
> - **🌪️ 시장 위기 리플레이**: 2020년 초단기 폭락, 2022년형 금리인상 약세장, 급등 후 폭락한 테마 자산,
>   2008년형 장기 금융위기 — 4가지 역사적 충격 패턴을 단순화한 가상 시나리오에서 하락장 한복판의
>   결정(손절/일부매도/보유/물타기)을 내려보고, "계속 보유"했을 때와 결과를 비교하며 행동재무학적
>   교훈을 얻습니다.
> - 위 3가지 모두 **실제 자금/모의투자와 독립된 학습 전용 기록**이며, 완료 조건에 따라 전용 뱃지·XP가
>   지급됩니다 (사기 방어 마스터, 청산의 교훈, 고배율 생존자, 위기 항해사 등).
>
> **이전 업데이트 (초기 자산 설정 + AI 기능 대폭 강화)**
> - **시작하기에서 초기 보유 자금/예적금을 직접 설정**할 수 있습니다. 처음 앱을 시작할 때 이미 갖고 있는
>   현금·예금 잔액과 가입해둔 예/적금 총액을 한 번에 입력해두면, 0원이 아니라 실제 내 자산 상태에서
>   바로 시작할 수 있습니다. 이후에도 사이드바의 **"잔액 직접 수정"**에서 언제든 보정할 수 있습니다.
> - **AI 종합 재무 리포트**: 재무 건강 점수·지출 예측·투자/저축 현황을 한 번에 묶어 Gemini가 마크다운
>   리포트로 작성해주고, `.md` 파일로 다운로드할 수 있습니다.
> - **AI 상담 챗봇 (멀티턴 대화)**: 1회성 진단을 넘어, 사용자의 최신 재무 데이터를 컨텍스트로 유지하며
>   자유 질의응답이 가능한 대화형 코치를 추가했습니다.
> - **재무 건강 점수 (규칙 기반 100점 만점)**: 저축 습관·비상금 여력·분산 투자·소비 통제·목표 관리 5개
>   항목을 투명하게 채점해, AI의 정성적 진단을 정량 지표로 보완합니다.
> - **다음 달 지출 예측**: 최근 3개월 가계부 데이터를 가중이동평균+추세로 분석해 다음 달 예상 지출과
>   급증(이상치) 여부를 미리 알려줍니다.
>
> **이전 업데이트 (실제자금/모의투자 분리 + 가계부 삭제 + 지속 로그인 안정화)**
> - **실제 자금(가계부·예/적금)과 모의투자를 완전히 분리**했습니다. 이전에는 하나의 `cash` 지갑을
>   가계부 지출·예적금 납입·모의투자 매수가 전부 같이 썼는데, 이제는 `real_cash`(실제 지갑)와
>   `mock_cash`(모의투자 전용 가상 시드머니)로 나눠서, 대시보드에도 "💰 실제 자금" / "🧪 모의투자(연습용)"
>   두 섹션으로 명확히 구분해 보여줍니다.
> - 가계부에 **수입 추가** 기능이 생겨서, 실제 지갑 잔액을 직접 채워나갈 수 있어요 (매일 쓰려면 필수 기능이었어요).
> - 가계부 지출/수입 기록에 **삭제(🗑️) 버튼**이 생겼습니다. 삭제하면 잔액도 자동으로 원복됩니다.
> - 지출 기록 시 "잔액 부족" 제한을 없앴습니다 — 실제 가계부는 잔액 통제 도구가 아니라 기록 도구이므로,
>   부족하면 마이너스로 표시되고 수입 추가로 채우는 방식이 더 현실적입니다. (예/적금·모의투자는 여전히 잔액만큼만 가능)
> - 사이드바에 **마지막 저장 시각/성공 여부**를 표시해서, DB에 매일의 기록이 실제로 저장되고 있는지 바로 확인할 수 있습니다.

사회초년생을 위한 **AI 소비·투자 코칭 모의 서비스**입니다. 실제 돈이 오가지 않는 교육용 시뮬레이션이며,
가계부 · 모의투자 · 예/적금 · 목표저축을 게임처럼 기록하면서 레벨/뱃지 시스템으로 재미를 더했습니다.

## 주요 기능
- **초기 자산 설정**: 시작 화면에서 이미 보유한 현금·예적금을 입력해 실제 상황에 맞춰 시작하고,
  이후에도 사이드바에서 잔액을 직접 보정할 수 있습니다.
- **투자성향 진단**: 10개 문항(투자기간·손실반응·목표·경험·소득·비상금·저축률·부채·목적·금융지식)을 바탕으로
  Gemini가 투자성향과 추천 자산배분을 진단합니다.
- **대시보드**: 순자산 추이, 자산 배분, 포트폴리오, 이번 달 지출, 뱃지 현황, **재무 건강 점수**,
  **다음 달 지출 예측**을 한눈에 봅니다.
- **모의투자**: 14개 가상 자산의 랜덤워크+뉴스 이벤트 시세, 종목별 뉴스, 실시간으로 갱신되는 모의 호가창(매수/매도 5호가)을 제공합니다.
- **가계부**: 카테고리를 골라 지출을 기록하고, 또래 평균 소비 비중과 비교합니다.
- **목표저축 / 예·적금**: 이름·시작일·기간·목표금액을 자유롭게 입력해 나만의 예/적금을 만들고,
  기간/금액 진행률을 "쫓아가는" 스타일의 진행 바로 확인합니다.
- **AI 코치**: 소비·투자·저축 데이터를 종합해 Gemini가 개인화된 진단과 실행 항목을 제시하고,
  이를 **다운로드 가능한 종합 리포트**로 확장할 수 있습니다.
- **AI 상담 챗봇**: 최신 재무 데이터를 컨텍스트로 유지하는 멀티턴 대화형 코치입니다.
- **재무 건강 점수**: 설명 가능한 규칙 기반 알고리즘으로 5개 항목을 채점해 100점 만점 점수를 제공합니다.
- **다음 달 지출 예측**: 가중이동평균+추세 기반으로 다음 달 지출과 급증 여부를 예측합니다.
- **🚨 리스크 체험관**: 금융사기 대응 훈련·레버리지 위험 시뮬레이터·시장 위기 리플레이로, 실제 돈을
  걸기 전에 위험을 먼저 겪어보는 학습 전용 공간입니다.
- **🧓 노후 준비 설계**: 은퇴 시점 자산 성장 곡선과 목표 노후자금(4% 룰 근사)을 비교하고, 부족분을
  메우는 데 필요한 월 저축액을 역산해줍니다.
- **🎭 소비 페르소나 & 📇 리포트 카드**: 가계부 데이터를 규칙 기반으로 분석한 소비 성향 태그와, 레벨·
  뱃지·재무 건강 점수·리스크 체험관 성적을 한 장으로 모은 다운로드형 요약 카드를 제공합니다.
- **📖 이용 가이드 & 첫 방문 튜토리얼**: 처음 오면 핵심 개념부터 전체 기능을 훑는 안내 화면이 자동으로
  뜨고, 이후엔 가이드 탭이나 사이드바 버튼으로 언제든 다시 볼 수 있어요.
- **레벨/뱃지 시스템**: 33종의 확정 지급형(확률 없음) 성취 뱃지로 좋은 습관을 게임처럼 쌓아갑니다.

## 실행 방법
```bash
pip install -r requirements.txt
streamlit run app.py
```

## 필요한 Secrets (선택)
`.streamlit/secrets.toml.example`을 참고해 `.streamlit/secrets.toml`을 만드세요.
- `GEMINI_API_KEY`: AI 코치 / 투자성향 진단 기능에 필요 (없으면 해당 기능만 비활성화되고 나머지는 정상 동작)
- `MONGO_URI`: 로그인 지속성(이름+출생연도+4자리 PIN 기반)에 필요 (없으면 세션 동안만 데이터 유지)

## 프로젝트 구조
```
app.py                 # 메인 Streamlit 앱 (UI/탭 구성, 리스크 체험관 UI 포함)
utils/config.py         # 자산/카테고리/뱃지/문항/사기 시나리오/위기 시나리오 등 정적 설정
utils/core.py           # 유저 상태, XP/레벨/뱃지, 시세+호가창 시뮬레이션, 목표/예적금, 리스크 체험관, 노후 설계, 소비 페르소나 로직
utils/database.py       # MongoDB 연동 (선택)
utils/ai_coach.py       # Gemini 기반 AI 코치 / 투자성향 진단
```

> ⚠️ 실제 시세·금융상품이 아닌 교육 목적의 가상 시뮬레이션입니다.
